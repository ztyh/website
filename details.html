Some details in programming

Flatten in numpy will take each row of the matrix and stack.
The vec operator in mathematics will take each column and stack.
The eig function in linalg will return the eigenvectors in the columns.

Julia
repmat takes a matrix and repeats it vertically by the first argument,
repeats it horizantally by the second argument.
If you had fed a vector it will be treated as a $n$ by 1 matrix.
mean with argument 1 takes the column wise mean.
mean with argument 2 takes the row wise mean.
I guess the number signifies along which dimension the mean is taken.
Similarly, sum with argument 1 takes the column wise mean.
sum with argument 2 takes the row wise mean.
elementwise division of an $n$ by $m$ matrix by a $1$ by $m$ matrix,
i.e., a transposed vector, returned element wise division of each row
by the 1 by $m$ matrix.
reshape also conforms with the vec operation in mathematics.
If one is deforming a vector into a matrix ($m$ by $n$),
we take consecutive $m$ elements and place them as columns from left to right.
If one is deforming a matrix into a 1 by $m\cdot n$ transposed vector,
the columns of the matrix are taken, transposed, and laid from left to right.
If one is deforming a matrix into a $m\cdot n$ by 1 vector,
the columns of the matrix are taken, and laid from top to bottom.
